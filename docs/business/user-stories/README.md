# User Stories

Agile User Stories für VegyScan, gruppiert nach Epics.

## Format

Alle Stories in [stories.md](./stories.md) im Format:

```
As a [persona],
I want to [action],
so that [benefit].
```

## Verknüpfung

Jede Story ist verknüpft mit:
- Use Cases (detaillierter Flow)
- Requirements (technische Details)
- Epics (thematische Gruppierung)
