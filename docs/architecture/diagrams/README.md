# Architecture Diagrams

Visual representations of VegyScan system architecture.

## Diagrams

- **[system-context.mmd](./system-context.mmd)** - C4 System Context: External dependencies
- **[components.mmd](./components.mmd)** - C4 Component: Internal structure
- **[data-flow.mmd](./data-flow.mmd)** - Data flow through the system

## Format

All diagrams use **Mermaid** syntax for version control and easy rendering in GitHub/IDEs.

## Viewing

- GitHub renders `.mmd` files automatically
- VS Code: Install "Mermaid Preview" extension
- Online: [Mermaid Live Editor](https://mermaid.live/)
