# Architecture Decision Records (ADRs)

Record of all significant architecture and technology decisions for VegyScan.

## What is an ADR?

An ADR documents a single architecture decision with its context, the decision itself, and consequences.

## Template

Use [0000-template.md](./0000-template.md) for new ADRs.

## Naming Convention

`NNNN-short-description.md`

**Example:** `0001-choose-react-native.md`

- **NNNN**: Sequential number, zero-padded to 4 digits
- **Description**: Lowercase, kebab-case, describes the decision
